# Sistema de Agendamento - Dashboard

## Introdução

A ideia de um sistema de agendamento surgiu de uma vontade própria nossa de aprender mais a linguagem de marcação (html) e estilo (Css) além de javascript para gerar a página futuramente. Basicamente a ideia é criar um sistema capaz de agendar consultas, ver horários livre e médicos especializados e a disposição para os clientes de uma Clínica fictícia.
